# Rückgewinnungsdepot

Das **Rückgewinnungsdepot** ist ein gemeinschaftliches System zum Spenden von Gegenständen. Sein Hauptzweck besteht darin, dass Spieler überschüssige Gegenstände loswerden können, die sie nicht brauchen, und gleichzeitig den gemeinsamen Belohnungspool füllen. Manchmal erhältst du dabei auch Gutscheine.

## Voraussetzungen

- Es gibt eine Levelanforderung, um Gegenstände zu spenden.
- Das Einlösen von Gutscheinen wird früher freigeschaltet als das Spenden.
- Prüfe die Spieloberfläche, wenn du dir nicht sicher bist, ob dein Charakter es schon nutzen kann.

## So funktioniert das Spenden

Wenn du Gegenstände spendest, werden sie aus deinem Inventar entfernt und dem gemeinsamen Belohnungspool hinzugefügt.

Nur bestimmte Gegenstände können gespendet werden, und einige davon müssen in festen Stapelgrößen gespendet werden. Das Spiel zeigt dir, was ausgewählt werden kann.

Manchmal erhältst du für das Spenden von Gegenständen einen Gutschein. Die Art des Gutscheins hängt vom gespendeten Gegenstand ab.
Anschließend kannst du den Gutschein verwenden, um Gegenstände aus dem gemeinsamen Pool einzulösen.

Spenden sind dauerhaft und können nicht rückgängig gemacht werden.

## Gutscheinarten

Im Rückgewinnungsdepot gibt es zwei Gutschein-Stufen.

### Niedriger Rückgewinnungsgutschein

Niedrige Gutscheine stammen normalerweise von häufigeren oder weniger wertvollen Spendengegenständen.

### Hoher Rückgewinnungsgutschein

Hohe Gutscheine stammen normalerweise von wertvolleren Spendengegenständen.

## So funktioniert das Einlösen

Du kannst einen Gutschein ausgeben, um eine zufällige Belohnung aus der entsprechenden Gutschein-Stufe zu erhalten.

Belohnungen stammen aus dem aktuellen gemeinsamen Pool, daher hängt dein Ergebnis davon ab, was Spieler gespendet haben. Gegenstände mit höherem Bestand im Pool werden mit größerer Wahrscheinlichkeit eingelöst. Die im Depot angezeigten Gegenstandsmengen sind ungefähre Werte und werden regelmäßig aktualisiert.
